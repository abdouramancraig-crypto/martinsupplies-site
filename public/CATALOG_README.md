# Catalog PDF Location

Place the catalog PDF file here with the exact filename:
**MartinSupplies_Catalog_Pro.pdf**

This file will be automatically downloaded when users submit the catalog request form on the products page.

## Current Setup:
- Path: `/public/MartinSupplies_Catalog_Pro.pdf`
- Download triggers after form submission
- Users receive the PDF plus notification signup

## To Add the PDF:
1. Copy `MartinSupplies_Catalog_Pro.pdf` to this `/public` folder
2. Commit and push to git
3. The download will work automatically
